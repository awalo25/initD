Node.js
Installation


npm


Installer avec npm:


npm install katex
# or globally
npm install -g katex



Fil


Installer avec Yarn:


yarn add katex
# or globally
yarn global add katex



Construire à partir de la source


Pour construire, vous aurez besoin de Git, Node.js 10 ou version ultérieure et de Yarn.


Clonez une copie du dépôt source GitHub :


git clone https://github.com/KaTeX/KaTeX.git
cd KaTeX



Installez ensuite les dépendances et exécutez le buildscript :


yarn
yarn build



Il transpilera automatiquement le code et n'inclura que les polices nécessaires pour les environnements cibles spécifiés par Browserslist config . Par exemple, si vous créez une application Web pour un kiosque avec Chrome 68, exécutez
 -la BROWSERSLIST="Chrome 68" yarn buildet elle produira une version sans transpilation, car elle prend entièrement en charge ES6 et n'inclut que les polices WOFF2.


Vous pouvez remplacer les polices incluses à l'aide de variables d'environnement. Définissez USE_(FONT NAME)
la variable d'environnement sur "true"ou "false", pour forcer l'inclusion ou l'exclusion d'un type de police, respectivement.`


Si vous souhaitez utiliser le KaTeX intégré dans d'autres projets, installez le package en spécifiant le chemin :


yarn add /path/to/KaTeX
# or using npm
npm install /path/to/KaTeX





Vous pouvez télécharger manuellement le package et le code source à partir des
 versions de GitHub .




Importation


KaTeX est exporté en tant que module CommonJS, qui peut être importé en utilisant require:


const katex = require('katex');



KaTeX exporte aussi conditionnellement un module ECMAScript :


import katex from 'katex';





Le module ES contient des syntaxes et des fonctionnalités ES6, et peut nécessiter une transpilation pour être utilisé dans d'anciens environnements.




Voir l' API pour les fonctions disponibles dans le katexmodule.


Inclure dans les pages Web


Pour rendre tout code HTML généré par exemple katex.renderToString, vous devrez toujours lier le fichier CSS, mettre les fichiers de police KaTeX à la disposition du client et utiliser le doctype HTML5. Voir l'utilisation du navigateur . Notez, cependant, que vous n'avez pas besoin d'inclure katex.jssur le client.


Utilisation de l'extension mhchem


L' extension mhchem 
ajoute ses fonctionnalités en modifiant le katexmodule. Vous pouvez donc utiliser KaTeX avec mhchem dans Node comme suit :


const katex = require('katex');
require('katex/contrib/mhchem'); // modify katex module
const html = katex.renderToString('\\ce{CO2 + C -> 2 C0}');
